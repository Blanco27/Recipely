Recipely · Settings — About Section

Three ways to say “About”

Each proposal sits at the bottom of the existing Settings screen, below Appearance & Language, and keeps the warm-editorial language already in the app — Paper panels, Fraunces titles, moss-green section labels. All three show version 1.0, a link to github.com/Blanco27/Recipely, and an open-source note. Pick one (or mix).

A List rows
Twin of the Language panel — “About” section label, a Paper panel with version + GitHub rows, quiet caption underneath. Most consistent, lowest effort.

Shared by all three
  • New “About” content at the bottom of the Settings screen.
  • Version pulled from BuildConfig.VERSION_NAME (currently 1.0) — no hardcoding.
  • GitHub row/button opens github.com/Blanco27/Recipely via an Intent (ACTION_VIEW).
  • Strings localized (EN/DE) like the rest of the screen.
Trade-offs
  • A — List rows: most consistent & cheapest; reuses the panel/row look 1:1.
  • B — Branded card: most character; adds an app mark & CTA button; tallest.
  • C — Quiet footer: calmest; great if About should feel like fine print.
